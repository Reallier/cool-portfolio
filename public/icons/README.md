## 图标来源建议

1. **简单Icons**: https://simpleicons.org/ - 包含大量技术品牌的SVG图标
2. **DevIcon**: https://devicon.dev/ - 专门的开发技术图标库
3. **官方网站**: 从各技术的官方网站下载官方logo
4. **Iconify**: https://icon-sets.iconify.design/ - 统一的图标API

## 图标规范

- 格式：SVG
- 尺寸：建议16x16或24x24
- 颜色：单色或原始品牌色
- 文件名：使用小写字母，多个单词用连字符分隔

## 暂时方案

如果暂时没有图标文件，组件会自动隐藏图标，只显示技术名称。
